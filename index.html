MyIO Application Prototype

Study of Mobile Phone Application Development

A dissertation submitted for the MSc in Software Engineering

by Cristina Messaggi de Souza

Start MyIO Final Version

The prototype was developed as a web application, running cross devices with a single implementation. WebSQL was used to handle the local data storage due to my familiarisation with SQL and currently broader browser capability. Handling of data needs to be more explored in future implementations as local storage can be turned off by the user (disabling the application to run) and due to data security (can be considered sensitive data). Another elements also need to be revisited, such as datatype validation and offline run.
